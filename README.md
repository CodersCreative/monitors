## Run The App
### Install
```
# To install from crates.io
cargo install monitors

# To run the installed program
monitors
```

### Build & Run
```
# Clone the repository:
git clone https://gitlab.com/officialccoders/monitors.git
cd monitors

# Build and run app with release tags:
cargo build --release
cargo run --release

# Or simply:
cargo run
```
